AFABL Zombie Game
=================

A simple game project to demonstrate NPCs controlled by AFABL agents.

Running
-------

.. code-block:: bash

  $ git clone git://github.com/csimpkins/afabl-zombie.git
  $ cd afabl-zombie/
  $ sbt update
  $ sbt update # yes, again - see https://github.com/philcali/sbt-lwjgl-plugin
  $ sbt run
